Allcomment＝（兵庫県 K.Y 24歳 ）

雨の日の歌 / たなひろ

現在2018年3月。Twitterでの独特なつぶやきは書籍化され、24ヶ国語に翻訳。世界中に感動と悲壮感が渦巻き、たなひろの入道雲が発生した。

あれから1年、ついに新曲発表の時が来た。

再生時間4分4秒、レンダリングソングの約8倍の長さのその歌は聞く人の心を何度もスパークリングし、たなひろ入道雲が積乱雲に変わる瞬間を貴方は目の当たりにする。
聴いて感じて欲しい。新たな時代の息吹を。

Virtual Tropical / tantan

聞く度に熱気と湿度を感じ、大量の汗が体から噴出し、舌にはパッ ションフルーツの幻味。 上司に何度も体調の異常を訴える程のトコナツMUSIC
だが安心して欲しい。この曲のタイトル。『Virtual Tropical』間違えがなければ、バーチャルと書かれている。

そう、現実ではないのだ。

この曲の作曲者tantanは音楽で人間の温度感覚を狂わせる魔術師。
現に電車内の女学生から自分から大量に滴り落ちる汗を指摘されるまで、私はtantanの魔法にかかっていることに気づく事ができなかった。
そんな魔術師tantanは1年前『女の子のパンティーの中に何が隠されているのか僕はまだ知らない』で一世を風靡した「僕たちに希望はあるズ」の元メンバーである
落ち着いて聞いて欲しい。そのメンバーの一人にあの『レンダリングソング』の「たなひろ」もいるのだ。
これは大変な化け物が出てきてしまった。左半身の振るえが止まらず、残り少ない有給を何度も消費してしまったものだ。
貴方はまんまとtantanの魔法にかかり、エアコンのスイッチに手を伸ばすだろう。

Ripple / ICCHI95

仮想通貨の名前ではない。non...this is MUSIC.
その波紋は聞く者全てを包み込むSpiritualな鎮魂曲。
賢者トリムタムの心の臓が響く限り、貴方の精神は眠りにつき、今までの罪は3Dの庭へ捧げられる...。
彼の音の響かせ方。胸の鼓動が速度を上げ、血圧が上がる。胡麻麦茶がいくつあっても足りない。
単調な音のレイヤー構想だと思いきや、そこに彼自身の人生談を感じさせる。
今回映像コンピに初参加となる『ICCHI95』
銅頭鉄額なメンバーが毎年集うこのコンピに単身乗り込んできた彼はいったい何者か。 強者たちに押しつぶされることなく自分を貫いて欲しい。彼の活躍にこれから目が離せない 。

要チェックや。

キラキラソング / Kashiwagi Masao

この曲で私は2回死んで、4回生き返っている。 ネタバレになってしまうが、 開始15秒で1度心肺停止 1分後息を吹き返す。
その後いつの間にか脳死している。
また1分半で脳が機能し始める。
瞬く間に前頭葉が次のステージへ、
ラストスパートで全身が覚醒し、母親(成美)の胎内へ戻る。
人の生き死にを自由に操るこの言葉の叫び。
死に癖がついてしまう為彼への深追いは禁物だろう。
因みに1年前の映像コンピでの『portable music』を聞いた父は甲子園へ一人アパートを借りて帰ってこなくなってしまった。

アンブル / Fumiya Tsutsu

今すぐ救急を呼ぶ準備をしたほうがいい。
この曲が終わるとき自我を忘れ『Fumiya Tsutui』について永遠とディグることになる。
人ごみの中から現れるRythm。
彼はその雑踏の断片を再構築し、正しき道へ直すスペシャリスト。
数々のアーティストに影響を与えたFumiya Tsutsui。
テンポ良くその磨かれた技を披露し、去年の『Chamber』と比べ向上心の現れが周りの曲より群を抜いているのが明らかだ。
不整脈をも起こしえるこのSound Trip 決して一人で聞いてはいけない。

サカナクションなんか目じゃないぜ。

海津大崎 雪景色/ Sugiyama Junichiro

聴く人みな声をそろえて言う、
「ももさん揃えて来たな」
琵琶湖の北端、海津湾の東に波食によって突き出た岩礁地帯「海津大崎」 桜名所でもあるこの地帯は絶景の雪景色としても名を轟かせている。
その景色を見に海外からの観光客も多い。
Sugiyama junichiro 通称　ももさん。
前回の映像コンピでも滋賀県の湖北を舞台にし、滋賀愛は他の追随を許さない。
バス釣りをしに琵琶湖へ立ち寄ることの多い彼ならでの表現であろう
この時期にマッチしたWinter song。訪れたことのない琵琶湖の湖上で数日過ごした気分にさせてもらえる。
この曲を聴いたが最後、老後になって孫に行った事もない滋賀についての昔話を語り始めること間違いなし。

こんなんなったの精華のせいか(GabberRemix) / higashno shigetoshi

精華のせいじゃない。これを聞いたお前が悪い。
京都精華大学の校歌になり損ね、残念な印象を持っている「こんなんなったの精華のせいか」をHigashino ShigetoshiがRemixを担当
原曲を一度バラし再構築しているが、元の素材を生かしつつ確実に彼自身の物となっている。
初めて聞いた時先生にやさしく怒られた時と同じ気持ちになり、興奮した。
低めのビート、さまざまなフィルターやエフェクトを駆使し聞く者をたっぷりと楽しませてくれて、生きている喜びを感じられる。
緊張感たっぷりからの「ｩｵｲ」は思わず笑みを浮かべてしまった。全身の毛を剃りたくなる。
最後の逆再生が何を意味しているかは自分の耳で確かめて欲しい。衝撃の事実が待っているだろう。
